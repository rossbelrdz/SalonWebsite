# Empty states

## Problema

Pantallas sin datos no deben verse “rotas”.

## Anatomía

- Ilustración o icono simple (un solo set)  
- Título corto (“Aún no hay citas”)  
- Descripción de una línea  
- CTA primario si aplica (“Agendar”, “Añadir servicio”)  

## Cuándo

- Listas admin vacías, mis citas, catálogo sin items, búsqueda sin resultados.

## No

- No mostrar tablas vacías sin mensaje.  
- No culpar al usuario.
