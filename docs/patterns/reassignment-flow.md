# Reasignación (3 opciones del cliente)

## Problema

Si se reasigna profesional/horario, el cliente elige sin fricción ni ambigüedad.

## Anatomía — 3 choice cards

1. **Aceptar** la reasignación propuesta  
2. **Reagendar** (otro slot / profesional)  
3. **Cancelar** (y política de reembolso si hubo prepago)

Clases mockup: `.choice-card`.

## Comportamiento

- Enlace único / vista “Reasignación pendiente”.  
- Notificación email/Telegram según matriz.  
- Prepago: candados y reembolso según F6.
